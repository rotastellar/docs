## Core Types

### Position

Geographic coordinates with altitude.

```python
from rotastellar import Position

# Kennedy Space Center
ksc = Position(latitude=28.5729, longitude=-80.6490, altitude_km=0.0)
```

### Orbit

Keplerian orbital elements with derived properties.

```python
from rotastellar import Orbit

orbit = Orbit(
    semi_major_axis_km=6778.0,
    eccentricity=0.0001,
    inclination_deg=51.6,
    raan_deg=100.0,
    arg_periapsis_deg=90.0,
    true_anomaly_deg=0.0
)

print(f"Period: {orbit.period_minutes:.1f} min")
print(f"Apogee: {orbit.apogee_km:.1f} km")
print(f"Perigee: {orbit.perigee_km:.1f} km")
```

### TimeRange

Time window specifications.

```python
from rotastellar import TimeRange
from datetime import datetime, timedelta

range = TimeRange(
    start=datetime.utcnow(),
    end=datetime.utcnow() + timedelta(hours=24)
)
```

## Client Configuration

```python
from rotastellar import RotaStellarClient

client = RotaStellarClient(
    api_key="rs_...",
    base_url="https://api.rotastellar.com/v1",  # Default
    timeout=30.0,     # Request timeout in seconds
    max_retries=3,    # Retry failed requests
)
```

## Error Handling

```python
from rotastellar.errors import (
    AuthenticationError,
    RateLimitError,
    NotFoundError,
    APIError
)

try:
    sat = client.satellites.get("INVALID")
except NotFoundError:
    print("Satellite not found")
except RateLimitError as e:
    print(f"Rate limited. Retry after {e.retry_after}s")
except AuthenticationError:
    print("Invalid API key")
```
