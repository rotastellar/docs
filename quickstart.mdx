# Quickstart

This guide will get you from zero to your first API call in under 5 minutes.

## 1. Get an API Key

<Card title="Request Early Access" icon="rocket" href="https://rotastellar.com/early-access/">
  Sign up to receive your API credentials.
</Card>

## 2. Install the SDK

<CodeGroup>
```bash Python
pip install rotastellar
```

```bash Node.js
npm install @rotastellar/sdk
```

```bash Rust
cargo add rotastellar
```
</CodeGroup>

## 3. Make Your First Request

<CodeGroup>
```python Python
from rotastellar import RotaStellar

client = RotaStellar(api_key="rs_your_api_key")

# Get ISS position
iss = client.intelligence.satellite("ISS")
pos = iss.position()

print(f"ISS Location: {pos.lat}, {pos.lon}")
print(f"Altitude: {pos.altitude_km} km")
```

```typescript Node.js
import { RotaStellar } from '@rotastellar/sdk';

const client = new RotaStellar({ apiKey: 'rs_your_api_key' });

// Get ISS position
const iss = await client.intelligence.satellite('ISS');
const pos = await iss.position();

console.log(`ISS Location: ${pos.lat}, ${pos.lon}`);
console.log(`Altitude: ${pos.altitudeKm} km`);
```

```rust Rust
use rotastellar::RotaStellar;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = RotaStellar::new("rs_your_api_key")?;

    // Get ISS position
    let iss = client.intelligence().satellite("ISS").await?;
    let pos = iss.position().await?;

    println!("ISS Location: {}, {}", pos.lat, pos.lon);
    println!("Altitude: {} km", pos.altitude_km);

    Ok(())
}
```

```bash cURL
curl https://api.rotastellar.com/v1/satellites/ISS \
  -H "Authorization: Bearer rs_your_api_key"
```
</CodeGroup>
