---
title: Negotiate OCU
api: POST /v1/ocu/negotiate
description: "Negotiates an Orbital Compute Unit allocation for a satellite workload"
---

<Note>
  **Base URL:** `https://rotastellar-cae.subhadip-mitra.workers.dev`
  — No API key required. CORS-validated.
</Note>

<RequestExample>
```bash
curl -X POST https://rotastellar-cae.subhadip-mitra.workers.dev/v1/ocu/negotiate \
  -H "Content-Type: application/json" \
  -H "Origin: https://rotastellar.com" \
  -d '{
    "satellite_id": "25544",
    "workload_power_w": 50
  }'
```
</RequestExample>
