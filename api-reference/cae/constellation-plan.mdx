---
title: Plan Constellation DAG
api: POST /v1/constellation/plan
description: "Generates an optimized execution plan (DAG) for a satellite constellation workload"
---

<Note>
  **Base URL:** `https://rotastellar-cae.subhadip-mitra.workers.dev`
  — No API key required. CORS-validated.
</Note>

<RequestExample>
```bash
curl -X POST https://rotastellar-cae.subhadip-mitra.workers.dev/v1/constellation/plan \
  -H "Content-Type: application/json" \
  -H "Origin: https://rotastellar.com" \
  -d '{
    "fleet": [
      {
        "norad_id": "99901",
        "name": "RS-LEO-1",
        "role": "compute",
        "orbit": { "altitude_km": 550, "inclination_deg": 53 }
      },
      {
        "norad_id": "99902",
        "name": "RS-LEO-2",
        "role": "compute",
        "orbit": { "altitude_km": 550, "inclination_deg": 53 }
      },
      {
        "norad_id": "99903",
        "name": "RS-LEO-3",
        "role": "aggregator",
        "orbit": { "altitude_km": 550, "inclination_deg": 53 }
      }
    ],
    "preset_id": "split-learning"
  }'
```
</RequestExample>
