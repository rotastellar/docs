---
title: Predict Hazards
api: POST /v1/hazards
description: "Predicts orbital hazards and generates a checkpoint schedule for a satellite"
---

<Note>
  **Base URL:** `https://rotastellar-cae.subhadip-mitra.workers.dev`
  — No API key required. CORS-validated.
</Note>

<RequestExample>
```bash
curl -X POST https://rotastellar-cae.subhadip-mitra.workers.dev/v1/hazards \
  -H "Content-Type: application/json" \
  -H "Origin: https://rotastellar.com" \
  -d '{
    "satellite_id": "25544",
    "prediction_hours": 24
  }'
```
</RequestExample>
