## Request

</ParamField>

<RequestExample>
```bash cURL
curl https://api.rotastellar.com/v1/runtime/jobs/job_xyz789 \
  -H "Authorization: Bearer rs_your_api_key"
```

```python Python
from rotastellar import RotaStellarClient

client = RotaStellarClient(api_key="rs_...")

# Get job status
job = client.runtime.job("job_xyz789")
print(f"Status: {job.status}")

# Wait for result
result = job.result(timeout=30)
print(result.text)
```

```typescript Node.js
import { RotaStellarClient } from '@rotastellar/sdk';

const client = new RotaStellarClient({ apiKey: 'rs_...' });

const job = await client.runtime.job('job_xyz789');
console.log(`Status: ${job.status}`);

const result = await job.result({ timeout: 30000 });
console.log(result.text);
```
</RequestExample>
