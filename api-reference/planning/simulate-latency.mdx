---
title: Simulate Latency
api: POST /v1/planning/latency
description: "Predict network latency based on orbital geometry"
---

## Request

<ParamField body="orbit" type="string" required>
  Orbit specification: `LEO-550`, `MEO-8000`, `GEO`, etc.
</ParamField>

<ParamField body="ground_stations" type="array" required>
  List of ground station regions: `us-west`, `us-east`, `europe`, `asia`, etc.
</ParamField>

<ParamField body="include_isl" type="boolean" default="false">
  Include inter-satellite link routing
</ParamField>

<ParamField body="constellation_size" type="number">
  Number of satellites for ISL routing options
</ParamField>

<ParamField body="user_location" type="object">
  Specific user location: `{"lat": 37.77, "lon": -122.41}`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.rotastellar.com/v1/planning/latency \
  -H "Authorization: Bearer rs_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "orbit": "LEO-550",
    "ground_stations": ["us-west", "us-east", "europe", "asia"],
    "include_isl": true
  }'
```

```python Python
from rotastellar import RotaStellarClient
from rotastellar.types import Position

client = RotaStellarClient(api_key="rs_...")

source = Position(latitude=37.7749, longitude=-122.4194)  # San Francisco
dest = Position(latitude=51.5074, longitude=-0.1278)      # London

latency = client.simulate_latency(
    source=source,
    destination=dest,
    orbit_altitude_km=550,
    relay_count=2
)

print(f"Total latency: {latency['total_latency_ms']:.1f} ms")
```

```typescript Node.js
import { RotaStellarClient, Position } from '@rotastellar/sdk';

const client = new RotaStellarClient({ apiKey: 'rs_...' });

const source = new Position(37.7749, -122.4194); // San Francisco
const dest = new Position(51.5074, -0.1278);     // London

const latency = await client.simulateLatency({
  source,
  destination: dest,
  orbitAltitudeKm: 550,
  relayCount: 2
});

console.log(`Total latency: ${latency.total_latency_ms?.toFixed(1)} ms`);
```
