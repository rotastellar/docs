---
title: Get Satellite Latency
api: GET /v1/satellites/{satellite_id}/latency
description: "Calculate communication latency between satellite and a ground observer"
---

## Request

<ParamField path="satellite_id" type="string" required>
  NORAD catalog ID or common name (e.g., "25544" or "ISS")
</ParamField>

<ParamField query="lat" type="number" default="37.7749">
  Observer latitude in degrees
</ParamField>

<ParamField query="lon" type="number" default="-122.4194">
  Observer longitude in degrees
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.rotastellar.com/v1/satellites/25544/latency?lat=40.7128&lon=-74.0060" \
  -H "Authorization: Bearer rs_your_api_key"
```

```python Python
from rotastellar import RotaStellarClient

client = RotaStellarClient(api_key="rs_...")

# Calculate latency from New York
response = client.http.get("/satellites/25544/latency", {
    "lat": 40.7128,
    "lon": -74.0060
})

print(f"Round-trip latency: {response['direct_link']['round_trip_latency_ms']:.1f} ms")
print(f"Compared to terrestrial: +{response['comparison']['overhead_percent']:.0f}%")
```

```typescript Node.js
import { RotaStellarClient } from '@rotastellar/sdk';

const client = new RotaStellarClient({ apiKey: 'rs_...' });

const response = await client.http.get('/satellites/25544/latency', {
  lat: 40.7128,
  lon: -74.0060
});

console.log(`Round-trip latency: ${response.direct_link.round_trip_latency_ms.toFixed(1)} ms`);
console.log(`Compared to terrestrial: +${response.comparison.overhead_percent.toFixed(0)}%`);
```
