---
title: Get Session State
api: GET /v1/sessions/{sessionId}
description: "Returns the full state of a simulation session including satellite positions, subsystems, and events"
---

<Note>
  **Base URL:** `https://sim.rotastellar.com`
  — No API key required.
</Note>

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sessionId` | string | The session ID returned from Create Session |

<RequestExample>
```bash
curl https://sim.rotastellar.com/v1/sessions/ses_abc123
```
</RequestExample>
