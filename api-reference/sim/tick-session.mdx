---
title: Tick Session
api: POST /v1/sessions/{sessionId}/tick
description: "Advances the simulation by a given duration and returns the updated state"
---

<Note>
  **Base URL:** `https://sim.rotastellar.com`
  — No API key required.
</Note>

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sessionId` | string | The session ID returned from Create Session |

<RequestExample>
```bash
curl -X POST https://sim.rotastellar.com/v1/sessions/ses_abc123/tick \
  -H "Content-Type: application/json" \
  -d '{
    "duration_s": 60
  }'
```
</RequestExample>
