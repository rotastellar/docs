---
title: Get Satellite Position
api: GET /v1/satellites/{satellite_id}/position
description: "Get current position of a satellite"
---

## Request

<ParamField path="satellite_id" type="string" required>
  NORAD catalog ID or common name (e.g., "25544" or "ISS")
</ParamField>

<ParamField query="at" type="string">
  ISO 8601 timestamp for position calculation (default: now)
</ParamField>

<RequestExample>
```bash cURL
curl https://api.rotastellar.com/v1/satellites/25544/position \
  -H "Authorization: Bearer rs_your_api_key"
```

```python Python
from rotastellar import RotaStellarClient

client = RotaStellarClient(api_key="rs_...")

# Get current position
pos = client.get_satellite_position("25544")
print(f"ISS: {pos.latitude:.2f}, {pos.longitude:.2f}")
print(f"Altitude: {pos.altitude_km:.1f} km")

# Get position at specific time
from datetime import datetime
pos_future = client.get_satellite_position("25544", at_time="2026-01-21T18:00:00Z")
```

```typescript Node.js
import { RotaStellarClient } from '@rotastellar/sdk';

const client = new RotaStellarClient({ apiKey: 'rs_...' });

// Get current position
const pos = await client.getSatellitePosition('25544');
console.log(`ISS: ${pos.latitude.toFixed(2)}, ${pos.longitude.toFixed(2)}`);
console.log(`Altitude: ${pos.altitudeKm.toFixed(1)} km`);

// Get position at specific time
const posFuture = await client.getSatellitePosition('25544', '2026-01-21T18:00:00Z');
```
