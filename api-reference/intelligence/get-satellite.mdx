---
title: Get Satellite
api: GET /v1/satellites/{satellite_id}
description: "Retrieve information about a specific satellite"
---

## Request

<ParamField path="satellite_id" type="string" required>
  NORAD catalog ID or common name (e.g., "25544" or "ISS")
</ParamField>

<RequestExample>
```bash cURL
curl https://api.rotastellar.com/v1/satellites/ISS \
  -H "Authorization: Bearer rs_your_api_key"
```

```python Python
from rotastellar import RotaStellar

client = RotaStellar(api_key="rs_...")

iss = client.intelligence.satellite("ISS")
print(f"Name: {iss.name}")
print(f"Altitude: {iss.position().altitude_km} km")
```

```typescript Node.js
import { RotaStellar } from '@rotastellar/sdk';

const client = new RotaStellar({ apiKey: 'rs_...' });

const iss = await client.intelligence.satellite('ISS');
const pos = await iss.position();
console.log(`Altitude: ${pos.altitudeKm} km`);
```
</RequestExample>
